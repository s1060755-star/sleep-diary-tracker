# Sleep Diary Tracker (Щоденник сну) 🌙

Консольна утиліта для обліку тривалості сну. Варіант №24.

## 📋 Функціонал

1. Додати запис сну - вказати дату, час засинання та пробудження
2. Переглянути всі записи - список усіх збережених даних
3. Середня тривалість за тиждень - аналіз останніх 7 днів
4. Найкоротша ніч за тиждень - визначення мінімальної тривалості

## 🚀 Як запустити
python main.py

## 📁 Формат даних

Дані зберігаються у файлі sleep_data.json у форматі:
[
  {
    "date": "2025-06-15",
    "sleep_time": "23:00",
    "wake_time": "07:00",
    "duration_hours": 8.0
  }
]
